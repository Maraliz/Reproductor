# Reproductor 

- Reescrito con es6 
- una base de datos  por medio de json-webserver
- con llamadas ajax
- css modular 
- dos imágenes por canción para poder usar el panel central


## instalacion

- clonar repo  ```git clone```
- correr json-server ```json-server db.json```
- Abrir index.html
